# 🚀 Crypto Sniper Bot V5.2: Ultimate Analytics

**Crypto Sniper Bot** — це інтелектуальна торгова система для біржі **KuCoin**, що працює на стратегії імпульсних пробоїв (**Breakout**) з багаторівневою фільтрацією ринкового шуму.

---

## 🛠 Технологічний стек
* **Мова:** Python 3.10+
* **Бібліотеки:** `pandas`, `numpy`, `ccxt`, `pyTelegramBotAPI`
* **База даних:** SQLite3 (локальне зберігання історії та аналітики)

---

## 📈 Ключові стратегії та фільтри

### 1. Сигнал на вхід (Breakout)
Бот відстежує локальні максимуми та мінімуми за останні **50 свічок** на 5-хвилинному таймфреймі. Вхід відбувається лише при пробої цих рівнів.

### 2. Детектор тренду (ADX)
Щоб уникнути "пилки" у флеті, впроваджено індикатор **ADX**:
* **ADX > 22**: Бот активний (сильний тренд).
* **ADX <= 22**: Бот в режимі очікування (флет).

### 3. Підтвердження об'ємом
Пробій вважається істинним лише якщо поточний об'єм торгів перевищує середній за 20 періодів у **1.5 рази**.

---

## 📊 Аналітичні можливості
Бот автоматично збирає статистику та дозволяє аналізувати ефективність:
* **Weekdays vs Weekends**: Порівняння профіту в робочі та вихідні дні.
* **Hourly Heatmap**: Виявлення найбільш прибуткових годин для кожної монети.
* **Daily Performance**: Детальний звіт по кожному дню тижня окремо.
