# 🚀 Crypto Sniper Bot V5.2: Ultimate Analytics

**Crypto Sniper Bot** — це інтелектуальна торгова система для біржі **KuCoin**, що спеціалізується на стратегії імпульсних пробоїв (**Breakout**) з використанням фільтрації за силою тренду (ADX) та об'ємом.

---

## 🛠 Технологічний стек

* **Мова:** Python 3.10+
* **Бібліотеки:** `pandas`, `numpy`, `kucoin-python`, `pyTelegramBotAPI`
* **База даних:** SQLite3 (локальне зберігання історії та аналітики)

---

## 📈 Ключові стратегії та фільтри

### 1. Сигнал на вхід (Breakout)
Бот відстежує локальні максимуми та мінімуми за останні **50 свічок** на 5-хвилинному таймфреймі. Вхід відбувається лише при пробої цих рівнів.

### 2. Детектор тренду (ADX)
Щоб уникнути хибних входів у "боковику", впроваджено індикатор **ADX**:
* **ADX > 22**: Бот активний (підтверджений тренд).
* **ADX <= 22**: Бот в режимі очікування (флет).

### 3. Підтвердження об'ємом
Пробій вважається істинним лише якщо поточний об'єм торгів перевищує середній за 20 періодів у **1.5 рази**.

---

## 📊 Аналітичні можливості
Бот автоматично збирає статистику кожної угоди та аналізує ефективність за допомогою Telegram-команд:
* **Weekdays vs Weekends**: Порівняння профіту в робочі та вихідні дні.
* **Hourly Heatmap**: Виявлення найбільш прибуткових годин для входу в ринок.
* **Daily Performance**: Детальний звіт по кожному дню тижня окремо.

## ⌨️ Команди Telegram

| Команда | Функція |
| :--- | :--- |
| `/status` | Поточні активні позиції та їх реальний PnL |
| `/stats` | Загальний вінрейт, кількість угод та чистий прибуток |
| `/report` | Порівняльний звіт (Прибуток у Будні vs Вихідні) |
| `/days` | Детальна статистика по днях тижня (Пн-Нд) |
| `/history` | Список останніх 10 закритих угод |
| `/check` | Перевірка стану API та з'єднання з біржею |
